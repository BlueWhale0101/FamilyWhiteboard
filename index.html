Quick Add Guide

  • Type a task and tap + or press Enter.
  • If you do not type a name, the Wes / Skye toggle decides the owner.
  • If you type Wes or Skye, that overrides the toggle.
  • The parser can pull due phrases like today, tomorrow, tonight, morning, afternoon, weekdays, this week, next week, and combinations like Tuesday morning or Friday afternoon.
  • Tap the mic button to dictate a task and auto-add it on supported browsers. Common filler like “remind me to” or “we need to” is cleaned up automatically, and simple helper phrases like “needs to” are stripped from the final task title, and an undo button appears for a few seconds.
  • Add a simple tag with a hashtag, like #party or #shopping. The tag is stored separately and removed from the task title.
  • You can also edit a tag in the task details view and filter the board by existing tags.
  • Recurring tasks can be set in the task details view: daily, weekly, weekdays, or monthly. When you complete one, the next one is created automatically.
  • Swipe left on a card to move it to the next basket.
  • Swipe right on a card to open edit.

Welcome to Garden Board
This is a garden whiteboard for Skye. The goal is simple: write things down fast, see what matters now, and keep the rest calm.
Start at the top
Use the quick add box for almost everything. Type a short phrase like “Skye groceries tomorrow” or “Wes bins Tuesday morning” and tap the plus button.
Check Today spotlight
If something is due today, it shows up in the spotlight section so it is easier to notice without scanning the whole board.
Use baskets
Upcoming is open by default. The other baskets can stay closed until needed. Swipe left on a task to move it to the next basket, swipe right to edit it, or tap the mic button in quick add to dictate a task.
Use archive for memory
Archive done tasks when you want a clean board, then show archived later to review what you did for projects like parties, holidays, or travel prep.
