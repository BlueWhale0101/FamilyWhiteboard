Quick Add Guide

  • Type a task and tap + or press Enter.
  • If you do not type a name, the Wes / Skye toggle decides the owner.
  • If you type Wes or Skye, that overrides the toggle.
  • The parser can pull due phrases like today, tomorrow, tonight, morning, afternoon, weekdays, this week, next week, and combinations like Tuesday morning or Friday afternoon.
  • Everything left over becomes the task title.
Skye groceries tomorrow
Wes bins Tuesday morning
laundry for Skye Friday afternoon
plan outing this weekend
